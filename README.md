## OpenSprinkler Fork

This fork of [OpenSprinkler-Firmware](https://github.com/OpenSprinkler/OpenSprinkler-Firmware) adds a simple NetworkMonitor service and script that checks for connectivie to google DNS. If connectivity fails, then the script reboots the system.

This is designed to help recover from network failures.
